# Baseline Machine Learning Model

## Model Architecture
This baseline model uses traditional machine learning techniques for text classification.

Architecture:
- TF-IDF Vectorization
- Classification model (e.g., Logistic Regression or Naive Bayes)

## Techniques Applied
- Text preprocessing (cleaning, tokenization)
- TF-IDF feature extraction
- Supervised classification
- Model training using scikit-learn

## Results
- Provides a baseline for comparison with deep learning models
- Faster training but generally lower performance than LSTM and CNN
